# HDR UK Events List

Used to track number of user registrations at events

![Events List](./images/events-list.png)

Backend API in Laravel

Frontend UI in NextJS

## Installation

**Backend**

```
cd backend
./install_and_run.sh
```

Open a new terminal

**Frontend**
```
cd frontend
./install_and_run.sh
```

Navigate to http://localhost:3000
